# /designer — DEPRECATED

This command has been replaced by `/frontend-design`, which provides 14 specialized modes with anti-slop design principles.

**Redirect:** Run `/frontend-design $ARGUMENTS` instead.

## Migration

| Old usage | New usage |
|-----------|-----------|
| `/designer build a login form` | `/frontend-design build a login form` |
| `/designer review the sidebar` | `/frontend-design critique sidebar` |
| `/designer animate the nav` | `/frontend-design animate nav` |
| `/designer` (no args) | `/frontend-design` |

## Available Modes

build, teach, polish, audit, critique, harden, animate, typeset, colorize, arrange, bolder, quieter, distill, overdrive

See `/frontend-design` for full documentation.
